## 0.0.1

### Added
- **FlutterishStepper**: A comprehensive stepper component for multi-step workflows
  - Support for both vertical and horizontal layouts
  - Multiple step states: indexed, complete, error, disabled, editing
  - Comprehensive theming support via StepperTheme
  - Built-in accessibility features with proper semantics 
  - Interactive step tapping with callbacks
  - Customizable icons, colors, and spacing
- **StepperStep**: Individual step configuration with title, subtitle, content, and state
- **StepperTheme**: Extensive theming system for visual customization
- Comprehensive test suite including widget tests and golden tests
- Complete example application demonstrating all features
- Full API documentation
